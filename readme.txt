

此次作业分为一下几个大的任务，每个大的任务下面还有小的任务

第一次任务分解
1、构建一个停车场
2、向空车场停车，如果停车成功，返回一个凭证，如果不成功，显示错误信息
3、向满的停车场停车，提示错误；
4、当停车场非空的时候，取车，那么如果凭证有效，并且与车是匹配的，那么取车成功；否则，显示错误信息
5、从一个空的停车场取车，提示错误。

第二次任务分解（停车仔）

1、构建一个停车场
2、向空车场停车，如果停车成功，返回一个凭证，如果不成功，显示错误信息
3、向满的停车场停车，提示错误；
4、当停车场非空的时候，取车，那么如果凭证有效，并且与车是匹配的，那么取车成功；否则，显示错误信息
5、从一个空的停车场取车，提示错误。

第三次任务分解

1、管理员查看停车仔信息
2、管理员自己停车，取车

第四次分解任务

1、主管查看信息
2、当主管查看信息出错时，提示错误
